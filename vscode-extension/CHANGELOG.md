# Change Log

All notable changes to the "leancontext" extension will be documented in this file.

## [0.1.0] - Initial Release
- Brand new extension: **LeanContext** (originally named AI Minify).
- **Chat Participant (`@lc`)**: Seamlessly intercept Copilot Chat prompts to inject optimized code.
- **Slash Commands**: Granular control over minification using `/all`, `/comments`, `/docs`, and `/deadcode`.
- **Token Analytics**: Live calculation of tokens saved via `gpt-tokenizer`.
- **Diff Preview**: Clickable `[View Optimized Code]` button to visualize exactly what was stripped.
- Bundled with `esbuild` for maximum performance and a tiny <2MB footprint.
